## About Me

I sit at the intersection of **web strategy, AI content design, and product thinking** — a practitioner who doesn't just write about AI but actively builds with it.

I spent 2.5 years as the **AI Chatbot Architect for the State of Connecticut**, designing NLP-driven conversation flows in Salesforce Einstein across multiple state agencies — reducing support wait times by 30% and increasing task completion rates by 25% for millions of public users.

Then I did something a little unusual: I **founded and shipped [Laffstr](https://laffstr.com)** — a full-stack SaaS platform for comedian career development — largely solo, using AI-augmented development workflows. That journey is what brought me to GitHub. What started as unfamiliar territory became a core part of how I build. Repositories, branches, pull requests, CI/CD deployment via Netlify, GitHub Copilot as a genuine accelerant — I use them all, and I'm not done learning.

A production web application I founded and built using AI-augmented development workflows — serving as sole founder, product strategist, content designer, and development QA.
